# Images Directory

## Default Images

### Deda's Photo (Get Well Card)

Please save your father's photo as `deda.jpg` in this directory.

**Requirements:**
- File name: `deda.jpg` (or `deda.png`)
- Recommended size: 800x1000 pixels (portrait orientation)
- The image will be automatically displayed in the middle of the Get Well Soon card

**To update the photo:**
1. Save your chosen photo as `images/deda.jpg`
2. The card will automatically load this image when the Get Well card is selected
3. You can still click on the image to change it temporarily

---

*Note: If the default image is not found, an upload button will be shown instead.*
